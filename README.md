# dtbf-site

Website for Dare to Break Free.

Built with Astro, Markdown content collections, and custom CSS.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Forms

The contact forms are wired for Formspree. Replace the placeholder form action in:

- `src/pages/contact.astro`
- `src/pages/share-your-story.astro`

with the live Formspree endpoint.
